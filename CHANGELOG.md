# Журнал изменений

Все значимые изменения проекта фиксируются в этом файле.

## [0.3.0] — 2026-07-12

### Добавлено

- раздел «Заметки»;
- создание, редактирование, поиск и удаление заметок;
- пять цветовых вариантов заметок;
- реактивный REST API `/api/notes`;
- таблица PostgreSQL `notes`;
- адаптивные карточки заметок.

### Изменено

- заголовок приложения учитывает файлы, закладки и заметки;
- блок статистики дополнен количеством заметок;
- интерфейс оптимизирован для трёх основных разделов.

## [0.2.0] — 2026-07-12

### Добавлено

- древовидное отображение папок;
- раскрытие и сворачивание папки кликом по её названию;
- рекурсивное отображение вложенных папок и файлов;
- отдельная кнопка «Открыть» для перехода в папку и загрузки файлов;
- индикатор загрузки ветки дерева;
- анимации раскрытия дерева.

### Исправлено

- запрос дочерних папок теперь использует параметр `parentId`;
- создание сущностей с заранее сформированным UUID выполняется через `R2dbcEntityTemplate.insert`;
- папки, файлы и закладки корректно сохраняются в PostgreSQL;
- сообщения backend и frontend переведены на русский язык.

## [0.1.0] — 2026-07-12

### Добавлено

- Spring Boot WebFlux backend на Java 21;
- React и TypeScript frontend;
- PostgreSQL в Docker Compose;
- создание и удаление папок;
- загрузка, скачивание и удаление файлов;
- локальное хранение содержимого файлов в `data/uploads`;
- создание, изменение и удаление закладок;
- перетаскивание файлов в область загрузки;
- уведомления и диалоги подтверждения;
- Framer Motion для анимаций;
- Lucide React для иконок;
- русский пользовательский интерфейс.
