# Журнал изменений

Все значимые изменения проекта фиксируются в этом файле.

## [Unreleased]

### Добавлено

- общая модель групп для закладок и заметок;
- REST API `/api/groups` для создания, переименования, получения и удаления групп;
- связь закладок и заметок с группой через `group_id`;
- запрет удаления непустой группы;
- переименование папок в интерфейсе файлового раздела;
- раскрываемые секции групп со счётчиками элементов;
- модальные окна создания и редактирования;
- меню `⋮` для папок, файлов, групп, закладок и заметок;
- оформленная зона выбора и перетаскивания файла;
- production-конфигурация Docker и Nginx для Tailscale;
- регистрация, подтверждение email и каждого входа через SMTP;
- вход, выход и смена пароля на Spring Security;
- серверные сессии и CSRF-защита;
- привязка всех пользовательских данных к владельцу;
- режим разработки без SMTP через `MAIL_ENABLED=false`.
- поиск файлов и папок текущего каталога;
- поиск закладок по названию, URL и описанию;
- изолированный `compose.orange-pi.yaml` с собственной PostgreSQL и Docker-сетью;
- пример окружения `.env.orange-pi.example`;
- полная инструкция `DEPLOYMENT_ORANGE_PI.md` для Orange Pi, Tailscale, VPS, nginx, HTTPS, резервного копирования и SSD.

### Изменено

- заголовок интерфейса показывает активный раздел;
- создание элементов доступно внутри выбранной группы;
- действия интерфейса объединены в адаптивные выпадающие меню;
- физические файлы хранятся под UUID-именами.
- production-контейнеры используют существующие PostgreSQL 15 и Docker-сеть `app-network`.
- текущий элемент пути в файловом разделе отображается как текст, а предыдущие элементы остаются ссылками для возврата.

### Исправлено

- сохранение и скачивание файлов с кириллицей в имени;
- создание папок с Unicode-именами;
- позиционирование выпадающих меню у границ окна.
- системный Basic Auth-диалог браузера заменён экраном входа приложения.
- healthcheck frontend на Orange Pi использует IPv4 `127.0.0.1` и корректно работает при IPv6-разрешении `localhost`.
- frontend nginx использует динамическое разрешение backend через Docker DNS и не падает, если backend запускается позже после перезагрузки платы.

## [0.3.0] — 2026-07-12

### Добавлено

- раздел «Заметки»;
- создание, редактирование, поиск и удаление заметок;
- пять цветовых вариантов заметок;
- реактивный REST API `/api/notes`;
- таблица PostgreSQL `notes`;
- адаптивные карточки заметок.

### Изменено

- заголовок приложения учитывает файлы, закладки и заметки;
- блок статистики дополнен количеством заметок;
- интерфейс оптимизирован для трёх основных разделов.

## [0.2.0] — 2026-07-12

### Добавлено

- древовидное отображение папок;
- раскрытие и сворачивание папки кликом по её названию;
- рекурсивное отображение вложенных папок и файлов;
- отдельная кнопка «Открыть» для перехода в папку и загрузки файлов;
- индикатор загрузки ветки дерева;
- анимации раскрытия дерева.

### Исправлено

- запрос дочерних папок теперь использует параметр `parentId`;
- создание сущностей с заранее сформированным UUID выполняется через `R2dbcEntityTemplate.insert`;
- папки, файлы и закладки корректно сохраняются в PostgreSQL;
- сообщения backend и frontend переведены на русский язык.

## [0.1.0] — 2026-07-12

### Добавлено

- Spring Boot WebFlux backend на Java 21;
- React и TypeScript frontend;
- PostgreSQL в Docker Compose;
- создание и удаление папок;
- загрузка, скачивание и удаление файлов;
- локальное хранение содержимого файлов в `data/uploads`;
- создание, изменение и удаление закладок;
- перетаскивание файлов в область загрузки;
- уведомления и диалоги подтверждения;
- Framer Motion для анимаций;
- Lucide React для иконок;
- русский пользовательский интерфейс.
